## :information_source: Sobre

Este aplicativo é o resultado de um projeto desenvolvido durante o trabalho de conclusão de curso do aluno BRAYAN ALVES MARTINS FERREIRA, ORIENTADO PELA DR. ZENAIDE CARVALHO DA SILVA. O objetivo desse App, é proporcionar uma abordagem diferenciada para aprendizagem dos estudantes de informática, tornando possível consultas rápidas de acordo com necessidades ou questionamentos momentâneos relacionados.
